== Description ==

WP-TagTip is an sexy way to show a related post of one specific tag with tooltips.

Please visit [the official website](http://www.coders.me/wordpress/wp-tagtip "WP-TagTip") for further details and the latest information on this plugin.

== Installation ==

1. Upload wp-tagtip directory to the /wp-content/plugins/ directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure and enjoy :)

Once the plugin is activated it creates a "WP-TagTip" panel under "Options". It will allow you to manage the TagTip options and styles.
